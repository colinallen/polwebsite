Using The Power of Logic Web Tutor Venn Diagram Component

Use the mouse to select which action you wish to perform, then mouse down on the diagram to perform the action. There are four actions available:
Shading
 Particular
Check
 Clear
  • Shading: Add or remove shading from the area of the diagram indicated by your mouse position.
  • Particular: Add or remove an X indicating the presence of a particular thing by pointing the mouse to an area or border where you wish the X to appear. (A syllogism with two particular premises will require two X marks.)
  • Check: Select this action and then mouse down anywhere on the diagram to find out whether you have correctly diagrammed the premises.
  • Clear: Select this action and mouse down anywhere on the diagram to return to the basic diagram with no shading or particulars marked.
